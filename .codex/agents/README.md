# Codex component briefs

Codex automatically follows the repository's nested `AGENTS.md` files. These briefs are copyable task prompts for three parallel worktrees or Codex sessions; they are not a claim of a separate runtime configuration format.

- `frontend.md`: UI and interaction lane
- `pipeline.md`: retrieval and orchestration lane
- `verification.md`: validation, evaluation, and MLE lane

Give each session one lane and a component-scoped branch. Shared model changes require cross-lane review.
