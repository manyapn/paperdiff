# Frontend lane

Own `apps/web/**`. Read the root and local `AGENTS.md` plus `docs/demo-script.md`. Edit the separated source files in `apps/web/src/`; do not hand-edit generated runtime payloads or create a parallel frontend. Render only real pipeline output without re-deriving classifications, and finish with `npm run lint && npm run test && npm run build` from `apps/web`.
